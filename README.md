# Порівняння алгоритмів пошуку підрядка

|----------|--------------------:|---------------------:|--------------------:|---------------------:|
| KMP | 0.0065 | 0.2799 | 0.0025 | 0.6797 |
| Боєр-Мур | 0.0016 | 0.0509 | 0.0014 | 0.1855 |
| Рабін-Карп | 0.0066 | 0.3501 | 0.0037 | 0.5811 |

## Висновки

**Найшвидший алгоритм — Боєр-Мур** для обох текстів і обох типів підрядків.

Різниця між алгоритмами найбільша при **вигаданому підрядку** — бо алгоритм змушений пройти весь текст до кінця не знайшовши збігу. Боєр-Мур у цьому випадку виграє найбільше, бо пропускає великі шматки тексту завдяки таблиці зсувів.

При **існуючому підрядку** всі алгоритми швидко знаходять збіг і зупиняються — тому різниця між ними менша.

**Загальний висновок:** Боєр-Мур є найефективнішим алгоритмом для пошуку підрядка як для коротких так і для довгих текстів.
